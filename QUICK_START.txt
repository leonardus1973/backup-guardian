SENSORI CREATI
================================================================================

sensor.backup_guardian_ultimo_backup
  - Mostra data/ora ultimo backup (timezone locale corretto!)
  - Attributi: nome, data, ora, dimensione, hash, tipo, destinazione

  - Numero totale backup
  - Attributi: lista completa backup con dettagli

sensor.backup_guardian_dimensione_totale
  - Spazio totale occupato in MB

RISOLUZIONE PROBLEMI
================================================================================

CARD NON APPARE?
  1. Svuota cache browser: Ctrl+Shift+R (pi√π volte!)
  2. Chiudi browser completamente e riapri
  3. Prova modalit√† incognito
  4. Verifica file esiste: /config/www/community/backup_guardian/

  1. Verifica URL risorsa corretto
  2. Svuota cache completamente (edge://settings/clearBrowserData)
  3. Riavvia Home Assistant

SENSORI VUOTI?
  1. Verifica di essere su HA OS o Supervised
  2. Controlla log: Impostazioni ‚Üí Sistema ‚Üí Log
  3. Cerca "backup_guardian"

  1. Verifica versione: deve essere v1.1.0+

  3. Verifica attributo backup_destination nei sensori
